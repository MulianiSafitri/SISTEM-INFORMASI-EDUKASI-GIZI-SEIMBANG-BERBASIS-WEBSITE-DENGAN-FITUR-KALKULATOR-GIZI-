# SISTEM-INFORMASI-EDUKASI-GIZI-SEIMBANG-BERBASIS-WEBSITE-DENGAN-FITUR-KALKULATOR-GIZI-
 Sistem Informasi Edukasi Gizi Seimbang adalah aplikasi berbasis website yang dirancang untuk membantu masyarakat memperoleh informasi gizi seimbang secara mudah dan akurat, sekaligus menyediakan fitur kalkulator gizi untuk menghitung kebutuhan nutrisi harian secara mandiri.
