# SISTEM-INFORMASI-EDUKASI-GIZI-SEIMBANG-BERBASIS-WEBSITE-DENGAN-FITUR-KALKULATOR-GIZI-
 Sistem Informasi Edukasi Gizi Seimbang adalah aplikasi berbasis website yang dirancang untuk membantu masyarakat memperoleh informasi gizi seimbang secara mudah dan akurat, sekaligus menyediakan fitur kalkulator gizi untuk menghitung kebutuhan nutrisi harian secara mandiri. Sistem ini bertujuan meningkatkan pemahaman, kesadaran, dan literasi gizi sebagai upaya mendukung kesehatan masyarakat.

Aplikasi ini menggabungkan konten edukasi gizi dengan teknologi digital sehingga pengguna tidak hanya menerima informasi teoritis, tetapi juga dapat mengetahui kebutuhan gizi sesuai kondisi tubuh masing-masing.

Fitur Utama
Administrator

Dashboard Admin
Menampilkan ringkasan data pengguna, artikel edukasi, dan aktivitas sistem.

Manajemen Konten Edukasi
Menambah, mengubah, dan menghapus artikel gizi seimbang.

Manajemen Data Gizi
Mengelola parameter kebutuhan gizi sebagai dasar perhitungan kalkulator.

Manajemen Pengguna
Mengelola akun pengguna yang terdaftar.

Monitoring Sistem
Melihat statistik penggunaan dan akses website.

Pengguna (Masyarakat)

Akses Edukasi Gizi
Membaca artikel gizi seimbang yang disajikan secara terstruktur.

Kalkulator Gizi Online
Menghitung kebutuhan kalori dan nutrisi berdasarkan usia, jenis kelamin, berat badan, tinggi badan, dan tingkat aktivitas.

Hasil Rekomendasi Gizi
Menampilkan estimasi kebutuhan energi harian sebagai panduan pola makan sehat.

Desain Responsif
Dapat diakses melalui perangkat desktop maupun mobile.
